PHOTO GALLI

An AI-powered image generator built with pure frontend web technologies. Type a prompt, get an image — it's that simple.

About the Generator

Photo Galli is an AI Image Generator Website developed using HTML, CSS, and JavaScript. It allows users to enter a text prompt and generate AI-based images through free API integration.

The system takes user input, sends it to AI image generation APIs (Puter.ai as primary, with Pollinations.ai as fallback), fetches the generated image in real-time, and displays it dynamically on the webpage. Users can save images to their personal gallery, download them, and view their prompt history.

This project demonstrates real-world application of frontend web development concepts such as DOM manipulation, API integration, asynchronous JavaScript, responsive UI design, and dynamic content rendering.

How It Works

01
Sign In — Create an account or use the demo credentials to access the generator.
02
Enter a Prompt — Describe the image you want in natural language. Be as descriptive as you like!
03
Generate — Click the Generate button. The app sends your prompt to Puter.ai for real-time AI image generation.
04
View & Save — The generated image appears instantly. Download it or save it to your personal gallery.
05
Browse Gallery — Visit the Photos page to see all your saved images in a responsive grid. Click any image to view it full-size with its prompt.

Technologies Used

HTML5 CSS3 JavaScript (ES6+) Bootstrap 5 Fetch API Async / Await DOM Manipulation localStorage CSS Flexbox CSS Grid Pollinations.ai API FLUX Model Stable Diffusion Responsive Design Media Queries

Features

  • User authentication with role-based access (User & Admin)
  • AI image generation via Pollinations.ai (FLUX, SDXL, Turbo models)
  • Sidebar prompt history (like ChatGPT / Grok)
  • Personal gallery — each user sees only their own images
  • Image modal with full-size view, prompt display, and date
  • Download generated images
  • Rate limiting — 6 free generations per user
  • Admin panel — manage users, view histories, set rate limits
  • Dark / Light mode toggle
  • Fully responsive design (mobile, tablet, desktop)
  • Retro pixel-art inspired UI theme
